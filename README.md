# Major Project

## Overview
**Major Project** is a web-based interactive application designed to simulate a question-answer interview loop. The application integrates text-to-speech, dynamic question generation, and user profile management to provide an engaging and adaptive interview experience.

---

## Features
- **User Registration & Profile Management:** Users can register, store their profiles locally, and resume their session seamlessly.  
- **Dynamic Interview Loop:** The system fetches questions based on selected topics and interacts with the user until the session ends.  
- **Text-to-Speech (TTS):** Questions are read aloud, enhancing accessibility and providing a realistic interview simulation.  
- **Topic Selection:** Users can select topics of interest to tailor the interview questions.  
- **Profile Statistics:** Tracks user activity and performance metrics for personalized feedback.  
- **Responsive Design:** Works across desktop and mobile browsers.

---

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Google Apps Script API  
- **Data Storage:** LocalStorage for user session persistence  
- **API Integration:** Fetch API for dynamic question generation and response handling
